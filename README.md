# Tableau de Bord Météo Dynamique

Projet réalisé dans le cadre du cours LOG3500 - Conception et programmation de sites Web I.

## Description

Cette application permet à l'utilisateur de rechercher une ville et d'obtenir les conditions météorologiques actuelles grâce aux API Open-Meteo.

## Fonctionnalités

- Recherche météo par ville
- Géocodage avec Open-Meteo
- Affichage du nom de la ville et du pays
- Température actuelle
- Vitesse du vent
- Statut météorologique
- Interface responsive
- Gestion des erreurs
- Accessibilité (ARIA)

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Async/Await
